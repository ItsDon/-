# 工作中常用git命令汇总


## git clone
>> git clone https://github.com/ItsDon/Git.git （在当前目录下创建Git目录）    
>> git clone  https://github.com/ItsDon/Git.git . (在当前目录下不创建新的目录)    
>> git clone https://github.com/ItsDon/Git.git  MyGit (在当前目录下创建MyGit目录)
